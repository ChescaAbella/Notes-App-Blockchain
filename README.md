# Notes App Blockchain

A simple **Notes Web Application** with authentication and CRUD (Create, Read, Update, Delete) features.  
Built for our project using:

- **Frontend:** React (Vite, JavaScript, CSS)
- **Backend:** Node.js (Express)
- **Database:** SQLite
- **Auth:** JWT + bcrypt
